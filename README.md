# Obesity Prediction with Research Paper – ML Web App

A Deep Learning–based **Machine Learning Web Application** that predicts obesity risk and provides **Explainable AI (XAI) insights** instead of black-box results.

This project transforms academic research into a real-world deployable solution, combining **ML modeling, Explainable AI, and a responsive React frontend.**

---

## 🌐 Live Application
**Web App:** https://obesity-prediction-research-paper.vercel.app/

---

## 📄 Research Foundation
This project is based on the published research paper:

**“Enhancing Obesity Prediction with Explainable AI: A DeepHealthNet-Based Comparative Study”**

The focus of the research was not only high prediction accuracy but also **transparency, interpretability, and ethical AI usage in healthcare.**

---

## 🚀 Features
- Real-time obesity category prediction
- Explainable AI output using SHAP / feature importance
- User-friendly health parameter input form
- Research-backed Deep Learning model
- Responsive UI with React + Tailwind
- Deployed and publicly accessible

---

## 🧠 Machine Learning Details
- Custom Neural Network (DeepHealthNet Concept)
- Trained on structured health dataset
- Balanced accuracy and interpretability focus
- Explainable AI integration for transparency
